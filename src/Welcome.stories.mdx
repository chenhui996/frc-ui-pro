# Frc-ui-pro

Frc-ui-pro 是基于 Ant Design 设计体系，二次封装的 React UI 组件库，主要用于研发企业级中后台产品。

## ✨ 特性

🌈 提炼自企业级中后台产品的交互语言和视觉风格。

📦 开箱即用的高质量 React 组件。

🛡 使用 TypeScript 开发，提供完整的类型定义文件。

⚙️ 全链路开发和设计工具体系。

## 安装

```shell
$npm install frc-ui-pro --save
```

## 引入样式文件

```ts
import "frc-ui-pro/dist/src/styles/global/index.css";
```
